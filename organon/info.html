Information About the Digital Book Display

This is a Flash-based flip-book. There is a navigation bar at the top of the page that allows you to do the following:
Display Multiple Pages: Shows pages in a left window pane and the book in a right window pane. To return to the full book view, select the cancel button at the bottom of the page.
Print: A dialog box will appear which allows to you print the left, right, or a range of pages.
Go To Any Page: Pages marked with an 'a' are interleaf pages which contain author's notes.
Navigation Arrows: First page, Previous page, Next page, Last page.
Zoom Tools: Zoom in, Zoom out, Reset to normal

You can begin turning pages by using the arrow keys in the navigation bar or by selecting one corner of the book and clicking.

You can zoom in or out on each page. After zooming, you can click-drag the book to move it around the screen. The reset button will return the book to its original size and position.

Throughout the book there are handwritten notes. Some of these notes are very complex and may have writing on both sides of the paper or have multiple pieces of paper pasted and folded together. On these pages, you'll see an . When you click the icon, the complex note will appear in a pop-up window. If the note has more then one page, please use the arrows on the navigation bar to move from one page to the next. The notes pages are displayed at a fixed width and if you enlarge, you simply need to click-drag the note to adjust the view. The quality of each note in the actual book varies by paper and ink quality as well as the size of the handwriting.

As an aside, the actual book has an interleaf between each set of pages. In many cases, there were no notations on the interleaf and we opted not to include those blank pages in this digital copy in hopes that the book may load faster without the additional files.
