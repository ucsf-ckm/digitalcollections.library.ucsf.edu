Information About the Digital Book Display

This is a Flash-based flip-book. There is a navigation bar at the top of the page that allows you to do the following:
Display Multiple Pages: Shows pages in a left window pane and the book in a right window pane. To return to the full book view, select the cancel button at the bottom of the page.
Print: A dialog box will appear which allows to you print the left, right, or a range of pages.
Go To Any Page
Navigation Arrows: First page, Previous page, Next page, Last page.
Zoom Tools: Zoom in, Zoom out, Reset to normal

You can begin turning pages by using the arrow keys in the navigation bar or by selecting one corner of the book and clicking.

You can zoom in or out on each page. After zooming, you can click-drag the book to move it around the screen. The reset button will return the book to its original size and position.
